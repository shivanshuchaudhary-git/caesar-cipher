#  Caesar Cipher (Python)

A simple **Caesar Cipher encryption & decryption tool** built using Python.
This program allows users to encode and decode messages by shifting letters of the alphabet.

##  Features

* Encrypt messages (encode)
* Decrypt messages (decode)
* Handles spaces and symbols
* Continuous run option (restart feature)
* Clean command-line interface

##  Tech Stack

* Python 3

##  Project Structure

* `main.py` → main program logic
* `art.py` → ASCII logo

##  How to Run

1️ Clone the repository

```bash
git clone <your-repo-link>
```

2️ Open the project folder

3️ Run the program

```bash
python main.py
```

##  How It Works

* User chooses **encode** or **decode**
* Inputs a message and shift value
* Program shifts each letter accordingly
* Displays the final encrypted/decrypted message
